# Quality Assurance

The Oregon Performance Audit Project is committed to producing accurate, transparent, and reproducible analyses.

## Quality Standards

Every recommendation is evaluated using standardized procedures documented in the Coding Manual.

Coding decisions should be supported by publicly available evidence whenever possible.

## Documentation

Each coding decision records:

- Evidence source
- Coding date
- Supporting notes
- Confidence level

## Consistency

Methodological changes are documented through version control.

Definitions remain standardized across all audits unless formally revised.

## Review

Future versions of OPAP may incorporate independent review of coded recommendations to further improve consistency and reliability.
